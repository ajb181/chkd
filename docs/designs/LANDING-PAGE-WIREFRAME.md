# chkd Landing Page Wireframe

Based on research of 100+ dev tool landing pages (Linear, Cursor, Vercel, Supabase) and 2026 SaaS trends.

---

## Research Insights Applied

### What Winners Do (Linear, Cursor, Vercel)
1. **Dark theme** - Developer tools live in dark mode
2. **Show don't tell** - Interactive demos/screenshots in hero
3. **Specific CTAs** - "Start building" not "Get started"
4. **Speed/simplicity messaging** - Lead with "fast" and "simple"
5. **Real UI** - Actual product screenshots, not abstract art
6. **Problem-to-solution arc** - Story in the hero
7. **Minimal but purposeful motion** - Subtle animations that demonstrate

### 2026 Trends to Apply
- Story-driven hero (show the transformation)
- Immersive product preview (interactive demo)
- Split-screen layouts (problem vs solution)
- Bold typography + personality
- Real customer contexts over abstract imagery

## Design Recommendations

### Typography
- **Headlines:** Bold, large, confident (consider Inter, Satoshi, or keep Space Mono for dev feel)
- **Body:** Clean sans-serif for readability
- **Code:** Monospace with subtle highlighting

### Color Palette (keep current dark theme)
- Background: Deep navy/black (#0a1628)
- Accent: Teal (#00d4aa) for CTAs, success states
- Secondary: Coral (#e59a7b) for warnings, highlights
- Text: Light gray (#e8f4f8) for readability

### Motion (Minimal + Purposeful)
- Hero: Subtle animation showing spec â†’ dashboard sync
- Progress bars: Animate on scroll
- Cards: Gentle hover lift
- NO flashy transitions, NO parallax

### Imagery
- REAL screenshots of the dashboard
- Terminal-style code blocks
- Workflow diagrams (keep clean, not busy)
- NO abstract 3D shapes or illustrations
